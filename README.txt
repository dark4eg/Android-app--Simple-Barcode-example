Android application example:
Application allows user to scan barcode using one of the barcode scanner apps found on the phone and it then performs a look up of the product using the
barcode. The lookup is done using www.upcdatabase.org. It only has about 1 million products and a simple to use api. While it only has 1 million products
which is limited it is perfect example of how you can use the scanned barcode result.
